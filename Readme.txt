Задача состояла в разработке решения для интернет-магазина "В один клик", направленного на увеличение покупательской активности постоянных клиентов путем персонализации предложений. Основные цели включали:

1) Построение модели для прогнозирования вероятности снижения покупательской активности клиента в следующие три месяца.

2) Выделение сегментов покупателей и разработку персонализированных предложений на основе данных модели и информации о прибыльности клиентов.

Для поиска лучшей модели были выполнены следующие шаги:

Создан пайплайн для предобработки данных, включающий обработку числовых и категориальных признаков.
Использован GridSearchCV для перебора гиперпараметров следующих моделей:

DecisionTreeClassifier
KNeighborsClassifier
LogisticRegression
SVC
